About the Dataset

The dataset is clean, has no missing values, and contains 30 features derived from cell nuclei measurements. These values are typically extracted using image analysis techniques and are not directly available in standard medical reports.

  • 357 benign cases (62.7%)
  • 212 malignant cases (37.3%)
  • Features include mean, SE, and worst values
  • Wisconsin Breast Cancer Dataset (UCI)
System Capabilities
